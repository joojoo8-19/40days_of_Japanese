## 1️⃣ 존재 표현: ~에 ~가 있습니다/없습니다 (あります/います)
### 💡 친절한 설명

| 문법 요소     | 일본어           |
|:----------|:--------------|
| **~에**    | 장소를 나타내는 조사 に |
| **~가**    | 주격 조사 が       |
| **~있습니다** | あります 또는 います   |

한국어의 '있다'에 해당하는 일본어 표현은 주어가 **생물**인지 **무생물**인지에 따라 다릅니다. 

| 형태 | 무생물       | 생물 | 한국어 의미 |
| :--- |:----------| :--- | :--- |
| **존재 긍정** | **あります**  | **います** | ~이/가 있습니다 |
| **존재 부정** | **ありません** | **いません** | ~이/가 없습니다 |

> 🌟 **핵심**: 장소를 나타내는 명사 뒤에는 항상 **に**를 붙이고, 주어 뒤에는 **が**를 붙여 **「場所 に 主語 が あります/います」** 형태를 만듭니다.

| 한국어 예문 | 일본어 문장 | 발음 (히라가나) |
| :--- | :--- | :--- |
| 가방 안에 지갑이 있습니다. | 鞄の中に 財布が あります。 | かばん の なか に さいふ が あります。 |
| 교실에 학생이 있습니다. | 教室に 学生が います。 | きょうしつ に がくせい が います。 |
| 공원에는 아무도 없습니다. | 公園には 誰も いません。 | こうえん に は だれ も いません。 |

## 2️⃣ 위치 표현: 앞, 뒤, 옆, 위, 아래, 안 (まえ, うしろ, よこ, うえ, した, なか)
### 💡 친절한 설명

사물의 위치를 나타내는 명사는 한국어와 마찬가지로 목적어 뒤에 붙여 사용하며, 이 뒤에 반드시 장소 조사 **に(니)**를 붙여서 '어디에'라는 의미를 완성합니다.

| 위치 명사 | 한국어 의미 | 예시 표현 | 히라가나 발음 |
| :--- | :--- | :--- | :--- |
| **前** | 앞 | 机の**前に** (책상 앞에) | つくえ の **まえ に** |
| **後ろ** | 뒤 | 車の**後ろに** (차 뒤에) | くるま の **うしろ に** |
| **上** | 위/위쪽 | テーブルの**上に** (테이블 위에) | テーブル の **うえ に** |
| **下** | 아래/밑 | ベッドの**下に** (침대 아래에) | ベッド の **した に** |
| **中** | 안/내부 | 箱の**中に** (상자 안에) | はこ の **なか に** |
| **横** | 옆/곁 | 銀行の**横に** (은행 옆에) | ぎんこう の **よこ に** |

> 🌟 **핵심**: **[기준 사물]** + **の** + **[위치 명사]** + **に** 순서로 구체적인 장소를 지정합니다.

| 한국어 예문 | 일본어 문장 | 발음 (히라가나) |
| :--- | :--- | :--- |
| 우체국은 역 앞에 있습니다. | 郵便局は 駅の**前に** あります。 | ゆうびんきょく は えき の **まえ に** あります。 |
| 제 가방은 의자 아래에 있습니다. | 私の鞄は 椅子の**下に** あります。 | わたし の かばん は いす の **した に** あります。 |
| 책상 옆에 강아지가 있습니다. | 机の**隣に** 犬が います。 | つくえ の **となり に** いぬ が います。 |

> 🌟 **참고**: よこ(横): 단순히 공간 상 옆에 위치함 vs. となり(隣): 인접함, 경계를 맞댄 옆을 의미함.

## 3️⃣ 의문 대명사: 언제, 어디, 누구, 어느 것 (いつ, どこ, だれ, どれ/どちら)
### 💡 친절한 설명

Day 1에서 배운 **何(なに/なん)** 외에도 '언제', '어디', '누구', '어느 것' 등 구체적인 정보를 묻는 의문 대명사들이 있습니다. 이 의문사들 뒤에는 정중한 의문 조사 **か(카)**를 붙여 문장을 완성합니다.

| 의문 대명사 | 한국어 의미 | 활용 예시 |
| :--- | :--- | :--- |
| **いつ** | 언제 | 誕生日 は **いつ** ですか。 (생일은 언제입니까?) |
| **どこ** | 어디 | 事務所 は **どこ** ですか。 (사무실은 어디입니까?) |
| **誰(だれ)** | 누구 | あの人 は **誰** ですか。 (저 사람은 누구입니까?) |
| **どれ** | 어느 것 (3개 이상 중) | あなたの 鍵 は **どれ** ですか。 (당신의 열쇠는 어느 것입니까?) |
| **どちら** | 어느 것 (2개 중) / 어느 쪽 | コーヒー と 紅茶、**どちら** が 良いですか。 (커피와 홍차, 어느 쪽이 좋습니까?) |

> 🌟 **핵심**: 의문 대명사는 문장 내에서 해당되는 명사 자리에 들어가며, 문장 끝에 **ですか**를 붙여 의문문을 만듭니다.

> **🌟 핵심2: どれ vs. どちら**
> * **どれ (도레):** **3개 이상**의 물건/선택지 중에서 고를 때 사용합니다.
> * **どちら (도치라):** **2개**의 선택지 중에서 고를 때 사용하며, '어느 쪽' 또는 '어디(위치)'의 정중한 표현으로도 쓰입니다.
<br/>

| 한국어 예문 | 일본어 문장 | 발음 (히라가나) |
| :--- | :--- | :--- |
| 일본으로 떠나는 것은 언제입니까? | 日本へ出発するのは **いつ** ですか。 | にほん へ しゅっぱつ する の は **いつ** ですか。 |
| 가장 가까운 편의점은 어디에 있습니까? | 一番近い コンビニは **どこに** ありますか。 | いちばん ちかい コンビニ は **どこ に** ありますか。 |
| 이 팀의 리더는 누구입니까? | このチームのリーダーは **誰** ですか。 | この チーム の リーダー は **だれ** ですか。 |

## 4️⃣ 지시 대명사: これ, それ, あれ, どれ, どちら

### 💡 친절한 설명

일본어의 지시 대명사(지물을 가리키는 말)는 거리 및 상황에 따라 **코(こ), 소(そ), 아(あ)** 세 가지 계열로 나뉩니다. **도(ど)** 계열은 의문사입니다.

| 구분 | 화자 가까이 (코) | 청자 가까이 (소) | 둘 다 멀리 (아) |
| :--- | :--- | :--- | :--- | 
| **물건 (것)** | **これ** (이것) | **それ** (그것) | **あれ** (저것) | 
| **위치 (곳)** | ここ (여기) | そこ (거기) | あそこ (저기) |
| **방향 (쪽)** | こちら (이쪽) | そちら (그쪽) | あちら (저쪽) |

| 한국어 예문 | 일본어 문장 | 발음 (히라가나) |
| :--- | :--- | :--- |
| 빨간색, 파란색, 노란색 중 당신 것은 어느 것입니까? | 赤、青、黄色の中で あなたのは **どれ** ですか。 | あか、あお、きいろ の なか で あなた の は **どれ** ですか。 |
| 도쿄와 오사카 중 어느 쪽이 좋습니까? | 東京と大阪、**どちら**が 好きですか。 | とうきょう と おおさか、**どちら** が すき ですか。 |
| '저것'은 저의 컴퓨터입니다. | **あれ**は 私の パソコンです。 | **あれ** は わたし の パソコン です。 |
