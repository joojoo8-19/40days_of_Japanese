# 🚀 일본어 회화패턴 41~50

## 41. ~했어요: 〜ました
### 41-1. 📢 예문
| 한국어 문장 | 일본어 문장 | 단어 | 문법 포인트 |
|:--- |:--- |:--- |:--- |
| 택시를 불렀어요 | タクシーを**呼びました**。 | 呼ぶ(よぶ): 부르다 | |
| 열쇠를 잃어버렸어요 | 鍵を**なくしました**。 | 鍵(かぎ): 열쇠, なくす: 잃어버리다 |
| 그녀는 요리 콘테스트에 나가서 우승 했어요 | 彼女は料理コンテストに出て**優勝しました**。 | 優勝(ゆうしょう)する: 우승하다 | |
| 전화 받았습니다(바꿨습니다) | お電話**かわりました**。 | かわる: 바뀌다 | |
| 신세졌습니다 | **お世話になりました**。 | お世話(せわ): 신세/보살핌 | |
| 골프 시합은 이겼어요? | ゴルフの試合は**勝ちました**か？ | 勝つ(かつ): 이기다 |
| 아니요. 졌어요. | いいえ、**負けました**。 | 負ける(まける): 지다 | |

### 41-2. 📚 문법 살펴보기
### 忘れる vs. なくす
| 단어 | 사용 | 예문 |
|:---- |:---- |:------- |
| 忘れる(わすれる) | 정보나 물건을 챙기는 것을 잊었을 때(두고 옴) |
| なくす | 가지고 있던 물건을 어디선가 잃어버려 행방을 모를 때 |

### 헷갈리기 쉬운 か~る 단어 정리

| 단어 (기본형) | 한자 표기 | 그룹 | 뜻 | 예문 활용 (ます형) |
| :--- | :--- | :---: | :--- | :--- |
| **かかる** | **掛かる** | 1그룹 | (시간/비용이) 걸리다 | 時間が**かかります** (시간이 걸립니다) |
| **かえる** | **帰る** | **1그룹(예외)** | (집으로) 돌아가다 | うちに**帰ります** (집에 돌아갑니다) |
| **かえる** | **変える** | 2그룹 | 바꾸다, 변화시키다 | 気分（きぶん）を **変えます** (기분을 바꿉니다 / 전환합니다)|
| **かわる** | **変わる** | 1그룹 | 바뀌다, 변하다| 信号(しんごう)が**変わります** (신호가 바뀝니다) |

## 42. 안 ~해요: 〜ません
### 42-1. 📢 예문
| 한국어 문장 | 일본어 문장 | 단어 | 문법 포인트 |
|:--- |:--- |:--- |:--- |
| 그 사람 몰라요 | その人を**知りません**。 | 知る(しる): 알다 |  |
| 회사 안 그만둬요 | 会社を**辞めません**。 | 辞める(やめる): 그만두다 |  |
| 오늘 오후에는 회의를 안 해요 | 今日午後は会議を**しません**。 | 会議(かいぎ): 회의 |  |
| 친구를 괴롭히지 않아요 | 友達を**いじめません**。 | いじめる: 괴롭히다 | |
| 인도에서는 돼지고기를 안 먹어요 | インドでは豚肉を**食べません**。 | 豚肉(ぶたにく): 돼지고기 | |
| 일본어를 알아요? | 日本語가 **わかります**か？ | わかる: 알다/이해하다 | 조사 が + わかる |
| 아니요. 전혀 몰라요 | いいえ、**全然わかりません**。 | 全然(ぜんぜん): 전혀 | |

### 42-2. 📚 문법 살펴보기
### 分かる(わかる) vs. 知る(しる) 
| 구분 | 分かる | 知る | 
|:--- |:--- |:--- |
| 핵심 의미 | 이해, 판명, 소통 | 지식/정보의 습득 및 소유 |
| 성격 | 내면의 사고 과정을 거쳐 '이해됨' (자동사) | 외부 정보를 머릿속에 '넣음' (타동사) |
| 조사 | 주로 が를 사용 (~가 이해되다) | 주로 を를 사용 (~를 알다) |
| 뉘앙스 | "내용을 파악했다", "말귀를 알아듣다" | "들어본 적 있다", "정보를 알고 있다" |
| 예문 | 彼の電話番号を知っていますか。| この数学の問題の答えがわかりますか。|
| 의미 | 그의 전화번호를 알고 있습니까? | 이 수학 문제의 답을 아나요? |

### 知る vs. する 활용
- **知る(1그룹)**: 知[り]ます/ 知[り]ません
- **する(3그룹)**: します / しません

## 43. 안 ~했어요: 〜ません でした
### 43-1. 📢 예문
| 한국어 문장 | 일본어 문장 | 단어 | 문법 포인트 |
|:--- |:--- |:--- |:--- |
| 사진을 안 찍었어요 | 写真を**撮りませんでした**。 | 撮る(とる): 찍다 | ます형 + ませんでした |
| 자기소개 안 했어요 | 自己紹介を**しませんでした**。 | 自己紹介(じこしょうかい): 자기소개 | |
| 선물 안 받았어요 | プレゼントを**もらませんでした**。 | もらう: 받다 | |
| 시험을 안 봤어요 | 試験を**受けませんでした**。 | 試験(しけん)を 受(う)ける: 시험을 보다 | |
| 관심을 안 가졌어요 | 興味を**持ちませんでした**。 | 興味(きょうみ): 관심, 持つ(もつ): 가지다 | | 
| 주말에 어딘가에 갔어요? | 週末どこか**行きました**か？ | どこか: 어딘가 |  |
| 아니요. 어디에도 안 갔어요 | いいえ、**どこにも行きませんでした**。 | どこにも: 어디에도 |  |

### 43-2. 📚 문법 살펴보기
### 조사 へ vs に vs で
| 조사 | 사용법 | 예문 | 
|:--- |:--- |:--- |
| へ(~으로) | 방향성 강조 | 南へ 行きます。(남쪽으로 갑니다) |
| に(~에) | 도착지 강조 / 장소에 단순히 존재하고 있을 때 | 部屋 います。 (방에 있습니다.) 駅に 着きました。 (역에 도착했습니다.) |
| で(~에서) | 장소에서 행동이 일어날 때 | 図書館で 勉強します。(도서관에서 공부합니다.) |

#### 💡 Tip: 
- に vs で: 똑같이 장소를 나타내지만, '있다(いる/ある)'와 같은 정적인 상태는 に, '공부하다, 먹다, 사다' 같은 움직임이 있는 행동은 で를 씁니다.  
- へ vs に: 현대 일본어의 이동 동사(가다, 오다, 돌아가다)에서는 혼용해서 쓰는 경우가 많지만, へ는 멀리 떠나는 느낌의 '방향'을, に는 딱 그 지점에 '도착'하는 느낌을 더 강하게 줍니다.

## 44. ~하지 마세요: [〜ない] + でください
### 44-1. 📢 예문
| 한국어 문장 | 일본어 문장 | 단어 | 문법 포인트 |
|:--- |:--- |:--- |:--- |
| 걱정하지 마세요 | **心配しないで**ください。 | 心配(しんぱい)する: 걱정하다 |  |
| 만지지 마세요 | **触らないで**ください。 | 触る(さわる): 만지다 |  |
| 여기에 쓰레기를 버리지 마세요 | ここにゴミを**捨てないで**ください。 | 捨てる(すてる): 버리다 | |
| 영화관 안에서 사진을 찍지 마세요 | 映画館の中で写真を**撮らないで**ください。 | 映画館(えいがかん): 영화관 | |
| 컴퓨터 전원을 끄지 마세요 | パソコンの電源を**切らないで**ください。 | 電源(でんげん): 전원 | |
| 다나카 씨, 너무 무리하지 마세요 | 田中さん、あまり**無理しないで**ください。 | 無理(むり)する: 무리하다 | |
| 항상 따뜻한 말 감사해요 | いつも**優しい言葉**、ありがとう。 | 優しい(やさしい): 다정한, 言葉(ことば): 말, 언어, 단어 | |

- **切る(きる)**: '자르다'가 기본형이지만, 전기/연결 등을 '끊다, 끄다'라는 의미로도 매우 자주 쓰입니다.

## 45. ~하지 않도록 (해): [〜ない] + ように（ね）
### 45-1. 📢 예문
| 한국어 문장 | 일본어 문장 | 단어 | 문법 포인트 |
|:--- |:--- |:--- |:--- |
| 절대 잊지 않도록 해 | 絶対**忘れないように**ね。 | 絶対(ぜったい): 절대 | 충고/금지 |
| 회식에서는 과음하지 않도록 해 | 飲み会では**飲みすぎないように**。 | 飲みすぎ: 과음 | |
| 내일 수업에 늦지 않도록 해 주세요 | 明日の授業に**遅れないように**してください。 | 授業(じゅぎょう): 수업 |  |
| 실패는 해도 후회는 하지 않도록 하고 있어 | 失敗はしても**後悔はしないように**している。 | 後悔(こうかい): 후회 | ~하도록 하고 있다 |
| 이제 더 이상은 살 찌지 않도록 운동을 시작 했어요 | もうこれ以上**太らないように**運動を始めました。 | 太る(ふとる): 살찌다 | |
| 갑자기 추워졌네 | 急に寒くなったね。 | 急に(きゅうに): 갑자기 | |
| 감기 걸리지 않도록 조심해 | 風邪を**引かないように**気をつけて。 | 風邪(かぜ)を引（ひ）く: 감기 걸리다 | |

### 45-2. 📚 문법 살펴보기
#### 문장 끝 て
문장 끝을 `~て`로 끝내면 `~て ください`의 `ください`가 생략된 **가벼운 명령이나 부탁**이 됩니다.

## 46. ~하지 않는 게 좋아: [〜ない] + 方（ほう）がいい
### 46-1. 📢 예문
| 한국어 문장 | 일본어 문장 | 단어 | 문법 포인트 |
|:--- |:--- |:--- |:--- |
| 그것은 안 만지는 게 좋아 | それは**触らない方がいい**。 | 触る(さわる): 만지다 | ~하지 않는 편이 좋다 |
| 저기는 위험하니까 안 가는 게 좋아 | あそこは危ないから**行かない方がいい**。 | 危ない(あぶない): 위험한 | |
| 벌레가 들어오니까 창문을 열지 않는 게 좋아 | 虫が入るから窓を**開けない方がいい**。 | 虫(むし): 벌레 | |
| 다른 사람 험담을 하지 않는 게 좋아 | 他人の悪口は**言わない方がいい**。 | 悪口(わるぐち): 험담 | |
| 남자친구에게 화장을 하지 않는 게 낫다고 들었어요 | 彼氏に化粧を**しない方がいいと言われました**。 | 化粧(けしょう): 화장 | 수동태 사용 |
| 나 몸이 안 좋아 | 僕、**調子が悪いんだ**。 | 調子(ちょうし) : 몸상태 | |

### 46-2. 📚 문법 살펴보기
### 聞く vs 言われる
- `聞く`는 내가 "정보로서 들었다"는 느낌이고, `言われる(수동형)`는 누군가가 나에게 "그렇게 말했다(충고/명령 등을 받았다)"는 뉘앙스입니다. 상대방의 직접적인 조언을 강조할 땐 `言われる`를 씁니다.

### から vs ので
- `から`는 원인이 나에게 있는 주관적 이유(내 생각에 ~하니까)를 나타냅니다. 
- 반면 `ので`는 사회적 통념이나 객관적 상황에 의한 이유(공손함)를 나타냅니다.

## 47. ~하지 않아서(원인) / ~하지 않고(상태): [〜な] + くて / [〜ない] + で
### 47-1. 📢 예문
| 한국어 문장 | 일본어 문장 | 단어 | 문법 포인트 |
|:--- |:--- |:--- |:--- |
| 밥을 먹지 않고 학교에 갔어 | ご飯を**食べないで**学校に行った。 | 먹지 않은 상태로 | 상태/수단 (~ないで) |
| 밥을 먹지 않아서 배가 고파 | ご飯を**食べなくて**お腹が空いた。 | 먹지 않았기 때문에 | 원인/이유 (~なくて) |
| 커피에 설탕을 넣지 않고 마셔요 | コーヒーに砂糖を**入れないで**飲みます。 | 砂糖(さとう): 설탕 | |
| 한자를 못 읽어서 시험에 떨어졌어 | 漢字が**読めなくて**試験に落ちた。 | 落ちる(おちる): 떨어지다 | |
| 어제 목욕하지 않고 자버렸어 | 昨日お風呂に**入らないで**寝ちゃった。 | お風呂に入る(おふろにいれる): 목욕하다 | |
| 일본 여행 어땠어? | 日本旅行、どうだった？ | | |
| 그게 말야, 일본어가 통하지 않아서 고생했어 | **それがね**、日本語が**通じなくて**苦労したよ。 | 通じる(つうじる): 통하다, 苦労(くろう): 고생, 수고 | |

## 50. ~하거라: [ます]형 어간 + 〜なさい
### 50-1. 📢 예문
| 한국어 문장 | 일본어 문장 | 단어 | 문법 포인트 |
|:--- |:--- |:--- |:--- |
| 여기에 이름을 쓰시오 | ここに名前を**書きなさい**。 | 書く -> 書き | 강한 명령/지시 |
| 운동장에 모이거라 | 運動場に**集まりなさい**。 | 集まる(あつまる): 모이다 | |
| 손을 깨끗이 씻어라 | 手をきれいに**洗いなさい**。 | 洗う(あらう): 씻다 | |
| 인간인 이상 자기 방은 스스로 치워라 | 人間(にんげん)である以上(いじょう):、自分の部屋は自分で**片付けなさい**。 | 片付(かたづ)ける: 치우다 | |
| 자기 자신을 가장 중요시 하거라 | 自分自身を一番**大切にしなさい**。 | 大切(たいせつ)にする: 소중히하다 | |
| 내일 조금 늦을 것 같아요 | 明日少し遅れそうです。 | ~そうです: ~할 것 같다 | |
| 되도록 빨리 오너라 | なるべく早く**来(き)なさい**。 | なるべく: 되도록 | 来る -> 来(き) |

### 50-2. 📚 문법 살펴보기
### 형용사를 부사로 만들기
| 구분 | 설명 | 예제 | 
|:--- |:--- |:--- |
| 나형용사 | 나형용사 어간에 `に`를 붙임 | `きれい(깨끗하다) + に = 깨끗하게` |
| 이형용사 | 어미 `い`를 떼고 `く`를 붙임 | `早い -> 早く(빠르게)`, `美味しい -> 美味しく(맛있게)` |
