#### ⚠️ 조사 「に」를 사용하지 않는 특정 시간 명사
조사 に는 같은 '~에'로서 시간을 나타내기도 합니다. 하지만 시간을 나타내는 모든 명사 뒤에 に를 덧붙일 수 있는 것은 아닙니다. 이걸 구분하려면, "**그 시간을 달력이나 시계 위에서 표현할 수 있는가?**"를 보면 됩니다. 

| に 사용여부 | 특징 | 예문 |
|:-- |:----- |:--------- | 
| O | 누구에게나 명확한 시간(구체적, 절대적) | 3時に (3시에), 曜日に (일요일에), 誕生日に (생일에) | 
| X | 말하는 시점에 따라 기준이 변하는 시간(추상적, 상대적) | いつ (언제), 今日(오늘), 明日(내일), 昨日(어제), 今週(이번 주), 毎日(매일) |

* Tip: '오늘'은 내일이 되면 '어제'가 되죠? 이렇게 기준이 바뀌는 단어들은 に라는 딱딱한 핀으로 고정할 수 없다고 생각하면 이해가 빠릅니다.

'오늘(今日)', '내일(明日)', '어제(昨日)', '매일(毎日)', '매주(毎週)', '아침(朝)', '오늘 아침(今朝)', '내년(来年)', '휴일(休日)' 등 **상대적인 시점이나, 빈도를 나타내는 명사**에는 「に」를 붙이지 않습니다. 이 명사들은 그 자체로 시간 정보를 포함하고 있어 「に」를 생략합니다.

* ❌ 今朝に 雪が 降りました。
* ⭕️ **今朝** 雪が 降りました。(오늘 아침 눈이 내렸습니다.) / **けさ** ゆきが ふりました。
* ⭕️ **毎日** 勉強します。(매일 공부합니다.) / **まいにち** べんきょうします。
* ⭕️ 火曜日に 会議が あります。(화요일에 회의가 있습니다.) / **かようびに** かいぎが あります。
* ⭕️ 1時30分に 駅で 待ちます。(1시 30분에 역에서 기다립니다.) / **いちじさんじゅっぷんに** えきで まちます。
* ⭕️ 12月1日に プレゼントを 買いました。(12월 1일에 선물을 샀습니다.) / **じゅうにがつ ついたちに** プレゼントを かいました。

### 3. 조사 「で」: '~으로, ~에서' (수단, 방법, 행위의 장소)

#### 3-1. 수단/방법
**용법:** 동작을 하는 데 사용하는 도구나 교통 수단, 언어 등을 나타냅니다.

1.  **電車で** 会社に 行きます。(전철로 회사에 갑니다.) / **でんしゃで** かいしゃに いきます。
2.  **日本語で** メールを 書きました。(일본어로 메일을 썼습니다.) / **にほんごで** メールを かきました。
3.  **携帯電話で** 写真を 撮ります。(휴대폰으로 사진을 찍습니다.) / **けいたいでんわで** しゃしんを とります。
4.  **ボールペンで** 名前を 書いてください。(볼펜으로 이름을 써주세요.) / **ボールペンで** なまえを かいてください。

#### 3-2. 장소를 나타내는 조사 「に」, 「で」의 차이점

**💡 친절한 설명** > * で는 어떤 동작이나 행위가 능동적으로, 구체적으로 일어나는 공간을 나타냅니다. 즉, '~에서 무엇인가를 했다'는 의미입니다.  
> * 반면 に는 행위가 없는 정적인 상태나 움직임의 도착점을 나타냅니다. 즉, '~에 존재한다' 또는 '~에 도착한다'는 의미입니다.  
> * 「で」와 「に」의 장소 용법 차이는 일본어 학습자들이 가장 헷갈려 하는 부분 중 하나이며, 문법적으로 엄격하게 구분됩니다.  

| 조사 | 장소에서의 역할 | 동사 유형 | 예문 | 히라가나 발음 |
| :---: | :---: | :---: | :---: | :---: |
| **に** | 정지된 상태의 **존재** 또는 **도착점** | ある/いる (있다), 行く/来る (가다/오다) | 部屋**に** います。(방에 **있다**) / 会社**に** 行きます。(회사에 **가다**) | へやに います。 / かいしゃに いきます。 |
| **で** | 동작/행위가 **일어나는 장소** | 食べる (먹다), 勉強する (공부하다), 会う (만나다) | 部屋**で** 食べます。(방에서 **먹다**) / 会社**で** 働きます。(회사에서 **일하다**) | へやで たべます。 / かいしゃで はたらきます。 |

1.  **図書館で** 勉強しました。(도서관에서 공부했습니다.) / **としょかんで** べんきょうしました。
2.  **レストランで** 友達と 会います。(레스토랑에서 친구와 만납니다.) / **レストランで** ともだちと あいます。
3.  **コンビニで** ジュースを 買いました。(편의점에서 주스를 샀습니다.) / **コンビニで** ジュースを かいました。
4.  私たちは **公園で** ゆっくり 遊びました。(우리는 공원에서 느긋하게 놀았습니다.) / わたしたちは **こうえんで** ゆっくり あそびました。

> ✏️ 추가 설명  
> 「図書館で 勉強しました。」 문장에서 「で」 대신 **「に」**를 사용해 **「図書館に 勉強しました。」**라고 하면 문법적으로 틀리고 원어민이 듣기에 매우 어색합니다. 동사 勉強する (공부하다)는 능동적이고 구체적인 행위를 나타내기 때문입니다.

### 4. 조사 「ぐらい」: 대략 (~정도)
**용법:** 시간, 수량, 정도의 **어림**을 나타냅니다.

1.  東京から 大阪まで **3時間ぐらい** かかります。(도쿄에서 오사카까지 **대략 3시간 정도** 걸립니다.) / とうきょうから おおさかまで **さんじかんぐらい** かかります。
2.  りんごを **4つぐらい** 買います。(사과를 **4개 정도** 삽니다.) / りんごを **よっつぐらい** かいます。
3.  毎日、**1時間ぐらい** 勉強します。(매일 **1시간 정도** 공부합니다.) / まいにち、**いちじかんぐらい** べんきょうします。
4.  私**ぐらい**の レベルの 人は たくさんいます。(저 **정도**의 레벨인 사람은 많이 있습니다.) / わたし**ぐらい**の レベルの ひとは たくさんいます。

### 5. 「〜から〜まで」: ~에서 ~까지
**용법:** 시간이나 장소의 **시작점(~부터)**과 **끝점(~까지)** 범위를 나타냅니다.

1.  **朝9時**から **夜5時**まで 働きます。(아침 9시부터 저녁 5시까지 일합니다.) / **あさくじ**から **よるごじ**まで はたらきます。
2.  この道は **駅**から **公園**まで 続きます。(이 길은 역부터 공원까지 이어집니다.) / このみちは **えき**から **こうえん**まで つづきます。
3.  授業は **月曜日**から **金曜日**まで です。(수업은 월요일부터 금요일까지입니다.) / じゅぎょうは **げつようび**から **きんようび**まで です。
4.  夏休みは **7月20日**から **8月31日**まで でした。(여름방학은 7월 20일부터 8월 31일까지였습니다.) / なつやすみは **しちがつはつか**から **はちがつさんじゅういちにち**まで でした。
