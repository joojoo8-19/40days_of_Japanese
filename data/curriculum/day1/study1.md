# 📚 Day 1. 40일 일본어 회화: 명사로 문장 만들기

## 1️⃣ 핵심 문장 구조: 「私は ~です」 (저는 ~입니다)
### 💡 친절한 설명

가장 기본이 되는 명사문 구조입니다. **명사**나 **な형용사** 뒤에 정중하게 끝맺는 표현인 「です」를 붙여 평서문을 만듭니다. 정중한 자리나 처음 보는 사람에게는 반드시 **です**를 사용해야 합니다. 

| 문법 요소 | 일본어 |
| :--- | :--- |
| **나** | 私 (わたし) |
| **~은/는** | は (わ) |
| **~입니다** | です (です) |

> 🌟 **핵심**  
> * 주격조사 「は」 발음은 ha가 아니라 wa(わ)로 합니다.
> * 「です」 앞에는 **명사**와 **な형용사**가 올 수 있습니다.
> * な형용사는 나중에 다루게되니, 지금은 명사에만 집중합시다! 

#### です 활용 (현재/과거, 긍정/부정)
| 형태 | 기본형 (반말) | 정중형 (입니다) | 한국어 의미 |
| :--- | :--- | :--- | :--- |
| **현재 긍정** | だ | **です** | ~이다 / ~입니다 |
| **현재 부정** | ではない | **じゃないです** | ~이 아니다 / ~이 아닙니다 |
| **과거 긍정** | だった | **でした** | ~이었다 / ~이었습니다 |
| **과거 부정** | ではなかった | **じゃなかったです** | ~이 아니었다 / ~이 아니었습니다 |

| 한국어 예문 | 일본어 문장 | 발음 (히라가나) |
| :--- | :--- | :--- |
| 저는 한국 사람입니다. | 私は 韓国人です。 | わたし は かんこくじん です。 |
| 오늘은 주말이 아닙니다. | 今日は 週末じゃないです。 | きょう は しゅうまつ じゃないです。 |
| 어제는 휴일이었습니다. | 昨日は 休みでした。 | きのう は やすみ でした。 |

## 2️⃣ 다양한 명사 연결 조사들
### 💡 친절한 설명

문장 속에서 명사와 명사를 연결하거나, 명사에 특정한 의미를 더해주는 다양한 조사들을 배웁니다.

| 구분 | 조사 | 한국어 의미 | 용법 |
| :--- | :--- | :--- | :--- |
| **소유/수식** | **の** | ~의 | 명사1이 명사2를 소유하거나, 명사1이 명사2를 수식 (例: 私のスマホ) |
| **강조/반복** | **も** | ~도, 또한 | 명사나 구에 동등한 자격이나 반복을 나타내어 「は」를 대신 (例: 私も行きます) |
| **나열/연결** | **で** | ~이고, ~하며 | 명사나 な형용사 문장을 긍정 형태로 나열하여 연결 (例: 彼は日本人で、親切です) |
| **나열 (모두)** | **と** | ~와, ~하고 | **모든** 사물이나 대상을 빠짐없이 **모두** 나열할 때 사용 (例: 犬と猫と鳥) |
| **나열 (일부)** | **や** | ~이나, ~등 | 사물을 **일부만** 예시로 들어 나열할 때 사용 (例: りんごやみかんなど) |

> 🌟 **핵심**  
> * で 연결조사의 な형용사 부분은 나중에 다루게되니, 지금은 명사에만 집중합시다! 

#### [の] 예문
| 한국어 예문 | 일본어 문장 | 발음 (히라가나) |
| :--- | :--- | :--- |
| 당신의 이름은 무엇입니까? | あなたの 名前は 何ですか。 | あなた の なまえ は なん ですか。 |
| 저것은 회사 건물입니다. | あれは 会社の建物です。 | あれ は かいしゃ の たてもの です。 |
| 저는 타나카 씨의 친구입니다. | 私は 田中さんの友達です。 | わたし は たなかさん の ともだち です。 |

#### [も] 예문
| 한국어 예문 | 일본어 문장 | 발음 (히라가나) |
| :--- | :--- | :--- |
| 저도 커피를 좋아합니다. | 私も コーヒーが好きです。 | わたし も コーヒー が すき です。 |
| 내일도 일입니까? | 明日も 仕事ですか。 | あした も しごと ですか。 |
| 이것도 주세요. | これも ください。 | これ も ください。 |

#### [で] 예문
| 한국어 예문 | 일본어 문장 | 발음 (히라가나) |
| :--- | :--- | :--- |
| 저는 학생이고, 아르바이트생입니다. | 私は 学生で、アルバイトです。 | わたし は がくせい で、 アルバイト です。 |
| 오늘은 맑아서 기분이 좋습니다. | 今日は 晴れで、気持ちがいいです。 | きょう は はれ で、 きもち が いい です。 |
| 그 영화는 조용하고, 멋졌습니다. | その映画は 静かで、素敵でした。 | その えいが は しずか で、 すてき でした。 |

#### [と/や] 예문
| 한국어 예문 | 일본어 문장 | 발음 (히라가나) |
| :--- | :--- | :--- |
| 물과 주스를 주문했습니다. | 水とジュースを 注文しました。 | みず と ジュース を ちゅうもん しました。 |
| 가방 안에 책이나 노트 **등**이 있습니다. | カバンの中に 本やノート**など**が あります。 | カバン の なか に ほん や ノート など が あります。 |
| 저는 어머니와 함께 여행했습니다. | 私は 母と 一緒に旅行しました。 | わたし は はは と いっしょ に りょこう しました。 |

> 🌟 **핵심**: 「など」는 ~등의 의미로 사용됩니다.

## 3️⃣ 조사 が(가) 와 か(카) 의 차이점
### 💡 친절한 설명

일본어 초급자가 가장 헷갈리는 두 조사의 핵심적인 역할에 대해 먼저 알아봅시다.

| 조사 | 역할 | 한국어 의미 |
| :--- | :--- | :--- |
| **が** | 1. 주격 조사 (이/가), 2. 목적격 (좋아함/싫어함/능숙함), 3. 문장 역접 (그러나) | ~이/가, ~을/를(일부), 그러나 |
| **か** | 1. 의문 조사 (문장 끝), 2. 부정칭 대명사 (어딘가, 누군가) | ~입니까?, ~니? |

> 🌟 **핵심**: 「が」는 주로 문장 안에서 **주어**나 **목적어(특정 동사)**를 표시하지만, 「か」는 **문장 끝**에 붙어서 의문을 만듭니다.

| 한국어 예문 | 일본어 문장 | 발음 (히라가나) |
| :--- | :--- | :--- |
| 돈이 필요합니다. | お金が 必要です。 | おかね が ひつよう です。 |
| 저는 한국 음식을 좋아합니다. | 私は 韓国料理が 好きです。 | わたし は かんこくりょうり が すき です。 |
| 혹시 누가 있습니까? | もしかして 誰か いますか。 | もしかして だれ か いますか。 |

## 4️⃣ 「何」의 발음 구분(なに vs. なん)
### 💡 친절한 설명
기본적으로 「何」은 なに 라고 발음하지만, 발음의 편리성 등과 같은 규칙에 의해 なん 으로 발음하기도 합니다.

| 구분 | 용법                                                                            | 예시 |
|:---|:------------------------------------------------------------------------------| :--- |
| なに | 대부분의 상황에서 사용되는 **기본 발음**입니다.<br/>특히 뒤에 '무엇'이라는 의미 자체를 강조하는 조사나 동사가 올 때 사용됩니다. | なにを 食べますか (무엇을 먹습니까?) |
| なん | の, と, で 등 자음 발음이 뒤따를 때, <br/> 時間(じかん), 歳(さい), 人(にん) 등 수량을 나타내는 단어가 올 때 사용됩니다. | なんですか (무엇입니까?), 何時 (なんじ, 몇 시) |

#### 예문
| 한국어 예문 | 일본어 문장 | 발음 (히라가나) |
| :--- | :--- | :--- |
| 이것은 무엇입니까? | これは 何ですか。 | これ は なん ですか。 |
| 당신은 누구입니까? | あなたは 誰ですか。 | あなた は だれ ですか。 |
| 이 근처에 편의점이 있습니까? | この近くに コンビニが ありますか。 | この ちかく に コンビニ が ありますか。 |
